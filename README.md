# Título do Projeto O Código Consciente

## 📒 Descrição
E-book sobre o papel e o impacto da inteligencia artificial no mundo do desenvolvimento de software


## 🤖 Tecnologias Utilizadas
- Gemini
- Nano Banana

## 🧐 Processo de Criação
Utilizei o "Gemini" para revisar minha pesquisa sobre IAs Generativas, estruturar o conteúdo e refinar minhas ideias. Foi utilizado o "Nano Banana" gerar a capa do nosso e-book. Para concluir, a montagem final foi feita no Libre Office, por ser uma ferramenta para edição de texto.

## 🚀 Resultados
O e-book apresenta uma reflexão sobre as IAs Genativas no desenvolvimento de softwares, e como podem ser uma ferramenta poderosa na.

## 💭 Reflexão (Opcional)
Criar o e-book foi uma experiencia interessante na descoberta do potencial de geração das IAs.
